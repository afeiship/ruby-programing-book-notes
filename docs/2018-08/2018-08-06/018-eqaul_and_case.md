# === 与 case 语句
```conf
在 case 语句中， when 判断值是否相等时，实际是使用 === 运算符来判断的。
左边是数值或者字符串时，== 与 === 的意义是一样的
除此以外， === 还可以与 =~ 一样用来判断正则表达式是否匹配
或者判断右边的对象是否属于左边的类，等等。
对比单纯的判断两边的值是否相等，=== 能表达更加 广义的“相等”。
```

### example:
```rb
p (/zz/ === "xyzzy")    #=> true
p (String === "xyzzy")  #=> true
p ((1..3) === 2)        #=> true
```
