# 第一章 RUBY 初探

## 页数：1 - 11

## hello-world
```rb
print("hello world!\n");
```

## description:
```conf
1. print 是一个方法。所谓方法，就是对象的行为。
2. 数值的加法或乘法运算、字符串的合并、某时刻一小时后或者一天后的计算等操作，都是通过执行方法来 实现。
3. print 方法会把作为参数传递过来的字符串，输出到控制台中
```

## 单引号 VS 双引号
```rb
## 单引号
print('Hello, \nRuby\n!\n')

## 双引号
print("Hello, \nRuby\n!\n")
```
