## puts 与 print 的区别：
```conf
puts 方法与 print 方法稍有区别
puts 方法在输出结果的末尾一定会输出换行符。
```

## p 方法：
```conf
无论使用 print 方法还是 puts 方法，输出数值 1 和字符串 "1" 时，结果都只是单纯的 1。
这样一来，我们就无法判断输出的结果到底是属于哪个对象。
这种情况下，p 方法可以很好地解决这个问题。
```
## p/puts 输出字符串
```rb
puts "100" #=> 100
puts 100 p "100" p 100
#=> 100 #=> "100" #=> 100
```
